About Type

Type comes from github.com/shadowfacts/type, which is a free and open source clone of typing.io, a typing practice site for programmers. You can checkout the original displaying site here: https://type.shadowfacts.net

In my first intention, I am happy to type something to read it, this is like copyinig practice by hand when we are pupils. It's relatively slow, but a good way to input something into the MIND with more attention.

All files for typing must be on GitHub, it can be code snippets, articles, documentations, etc., I myself have created a repository to store anything I want to read by typing: tarvos21/type.

Getting started:

  1. Add a repo
  2. Add a file
  3. Type
