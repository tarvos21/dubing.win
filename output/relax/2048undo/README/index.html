# 2048 Undo This is the MODDED version of 2048 that includes unlimited UNDO, which stores locally i.e. stays even if you refresh the webpage and it saves only VALID moves. You can play it on your phone via [http://np1810.github.io](http://np1810.github.io) ## Auto Script You can also try the this [2048 Auto](https://github.com/np1810/2048undo/blob/master/2048auto.js) script. It assists you to reach a certain score by doing up-right-up-right strategy. Copy the whole script in the Firebug Console and run it, have fun... ;) ### Screenshot
